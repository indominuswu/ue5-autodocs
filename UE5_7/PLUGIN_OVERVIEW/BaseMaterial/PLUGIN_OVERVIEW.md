# BaseMaterial Plugin Overview

## 1. What this plugin does

- Provides a shared repository of base material functions and content for building materials.
- Supplies reusable material function assets rather than code modules.
- Intended as foundational content for other materials across projects.

## 2. Key Modules

- No C++ modules; content-only plugin delivering material functions and assets.

## 3. Important Types & APIs

- Not applicable (content plugin). Material functions are provided as assets for use in the Material Editor.

## 4. Typical usage patterns

- Enable the plugin to make the base material function library available in the Material Editor.
- In material graphs, search for the provided BaseMaterial functions and include them as reusable building blocks.

## 5. Version-specific notes (UE 5.7)

- Beta content plugin enabled by default.
- No explicit UE 5.7-specific notes found; overview reflects the current asset set.
