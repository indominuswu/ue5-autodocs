# LiveLinkFaceImporter Plugin Overview

## 1. What this plugin does
- Imports CSV recordings generated by the Live Link Face iOS app.
- Converts captured facial animation data (including timecode) into Unreal assets via an editor factory.

## 2. Key Modules
- **LiveLinkFaceImporter** (Editor)  
  - Role: Provides an import factory for Live Link Face CSV files.
  - Notable types: `ULiveLinkFaceImporterFactory`.

## 3. Important Types & APIs

### `ULiveLinkFaceImporterFactory`
- Role: `UFactory` that validates and imports Live Link Face CSV files.
- Key behavior: Infers frame rate, parses timecode, creates LiveLink subjects from CSV headers, and surfaces tooltips in the import dialog.
- Functions override: `FactoryCanImport`, `FactoryCreateText`, plus helpers `LoadCSV`, `CreateSubjectString`, `ParseTimecode`.

## 4. Typical usage patterns
- Editor: Choose Import and select a Live Link Face CSV; the factory creates the corresponding asset/animation with subject data extracted from the file.
- Ensure timecode/frame rate are correctly parsed; the factory attempts to infer frame rate when not explicit.
