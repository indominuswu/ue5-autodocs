# PCG Biome Sample Plugin Overview

## 1. What this plugin does
- Provides sample content for the PCG Biome toolchain.
- Bundles example graphs/assets that rely on `PCGBiomeCore` and the base `PCG` plugin.
- No code modules; purely content and resources.

## 2. Editor/Runtime surfaces
- User-facing: Yes - Sample content/assets for PCG biome workflows.

## 3. Key Modules
- No code modules are declared. The plugin supplies content and depends on `PCGBiomeCore` and `PCG`.

## 4. Important Types & APIs
- None. This plugin ships sample assets only.

## 5. Typical usage patterns
- Enable after enabling `PCGBiomeCore` and `PCG`.
- Browse the included Content folder for example biome graphs and settings to copy into your project.

## 6. Version-specific notes (UE 5.7)
- Marked experimental in the `.uplugin`.
- No explicit UE 5.7-only behaviors noted; overview based on current assets.
