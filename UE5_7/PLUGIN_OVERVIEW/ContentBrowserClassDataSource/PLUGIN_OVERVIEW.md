# Content Browser - Class Data Source Plugin Overview

## 1. What this plugin does

- Provides a Content Browser data source that surfaces classes (UClass types) in the browser.
- Enabled by default to allow browsing class assets/types.

## 2. Editor/Runtime surfaces

- User-facing: Yes - Default Content Browser data source that surfaces `UClass` types in the browser so users can browse/search classes alongside assets.

## 3. Key Modules

- **ContentBrowserClassDataSource** (Editor, Default)  
  - Role: Supplies class data to the Content Browser data source framework.

## 3. Typical usage patterns

- Keep enabled to browse and search classes alongside assets within the Content Browser.
