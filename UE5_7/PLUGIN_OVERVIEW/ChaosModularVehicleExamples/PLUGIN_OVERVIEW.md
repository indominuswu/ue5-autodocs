# Chaos Modular Vehicle Examples Plugin Overview

## 1. What this plugin does

- Provides example assets for the Chaos Modular Vehicle framework.
- Supplies sample vehicle setups demonstrating how to assemble modular components.

## 2. Editor/Runtime surfaces
- User-facing: Yes - Ships example vehicle assets/content for teams to open and study in the editor.

## 3. Key Modules

- No modules are declared in the `.uplugin`; the plugin consists of example assets/content.

## 4. Important Types & APIs

- Uses the runtime components from `ChaosModularVehicle` and `ChaosModularVehicleEngine`; no additional code types are defined here.

## 5. Typical usage patterns

- Enable alongside `ChaosModularVehicle` to access sample vehicles and reference content.
- Open the example assets to see how modular vehicle components are combined and configured.

## 6. Version-specific notes (UE 5.7)

- Experimental example content, disabled by default in this UE 5.7 tree.
